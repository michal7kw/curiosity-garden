<h1>Anki</h1>
<h2>Overview</h2>
<p>Anki is a free, open-source flashcard program that uses <a href="/garden/dev_admin/concept/spaced-repetition/" class="internal-link">Spaced Repetition</a> to optimize memory retention. Unlike simple flashcard apps, Anki schedules reviews based on how well you know each cardâ€”showing difficult cards more often and easy cards less frequently. It's the gold standard tool for anyone serious about long-term memorization.</p>
<h2>Key Information</h2>
<ul>
<li><strong>Type:</strong> Software Tool</li>
<li><strong>Author:</strong> Damien Elmes</li>
<li><strong>First Released:</strong> 2006</li>
<li><strong>Platforms:</strong> Windows, Mac, Linux, iOS, Android, Web</li>
<li><strong>Cost:</strong> Free (iOS app paid, but AnkiDroid free)</li>
</ul>
<h2>Why This Resource?</h2>
<p>Anki implements <a href="/garden/dev_admin/concept/spaced-repetition/" class="internal-link">Spaced Repetition</a> algorithms that have been refined over decades of cognitive science research. It's used by medical students, language learners, and anyone who needs to memorize large amounts of information efficiently.</p>
<h2>Main Topics</h2>
<ol>
<li><strong>Spaced Repetition Algorithm</strong>: SM-2 based scheduling that optimizes review timing</li>
<li><strong>Card Types</strong>: Basic, cloze deletion, image occlusion, and custom types</li>
<li><strong>Deck Organization</strong>: Hierarchical organization of flashcard decks</li>
<li><strong>Sync</strong>: Cloud sync across all devices</li>
<li><strong>Add-ons</strong>: Extensive plugin ecosystem for customization</li>
</ol>

<h2>Summary</h2>
<h3>Core Message</h3>
<p>Make reviewing a daily habit and let the algorithm handle scheduling. Consistency beats intensity.</p>
<h3>Key Arguments/Points</h3>
<ol>
<li><strong>Daily reviews</strong>: 10-30 minutes daily is more effective than occasional cramming</li>
<li><strong>Make good cards</strong>: One fact per card, test understanding not recognition</li>
<li><strong>Active recall</strong>: The act of retrieving information strengthens memory</li>
<li><strong>Low-overhead learning</strong>: Once set up, the system maintains itself</li>
</ol>
<h3>Notable Features</h3>
<ul>
<li><strong>AnkiConnect</strong>: API for integration with other tools</li>
<li><strong>Cloze deletions</strong>: Fill-in-the-blank cards for learning in context</li>
<li><strong>Image occlusion</strong>: Hide parts of images (anatomy diagrams, etc.)</li>
<li><strong>Custom scheduling</strong>: Advanced users can modify the algorithm</li>
</ul>
<h2>My Notes</h2>
<h3>What I Learned</h3>
<p>The power of Anki comes from consistency and good card design. Bad cards waste time; good cards feel effortless. The <a href="/garden/dev_admin/course/learning-how-to-learn/" class="internal-link">Learning How to Learn</a> course taught me that cards should test <em>understanding</em>, not just surface recognition.</p>
<h3>How It Changed My Thinking</h3>
<p>Shifted my approach from &quot;learn and hope to remember&quot; to &quot;learn and systematically maintain.&quot; Every important concept I learn now gets a card.</p>
